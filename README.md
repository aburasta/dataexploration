# Data Exploration

A small collection of data-exploration notebooks, kept in GitHub so they are
accessible from any device (including this phone) even if a local machine is lost.

## Structure

```
.
├── README.md
└── notebooks/
    ├── iris_data_exploration.ipynb      # Iris flower dataset — load, shape, nulls, basic EDA
    └── titanic_data_exploration.ipynb   # Titanic dataset — load, shape, nulls, basic EDA
```

## Notebooks

| Notebook | Dataset | What it does |
|----------|---------|--------------|
| `notebooks/iris_data_exploration.ipynb` | Iris (UCI) | Loads the Iris dataset, inspects rows/columns, checks for missing values. |
| `notebooks/titanic_data_exploration.ipynb` | Titanic | Loads the Titanic dataset, inspects rows/columns, checks for missing values. |

## Working on your phone

Every file lives in this Git repository, so nothing depends on a single computer.
To keep any new work safe, **commit and push it** — anything that only lives on a
device's local disk is not backed up and can be lost with the device.
