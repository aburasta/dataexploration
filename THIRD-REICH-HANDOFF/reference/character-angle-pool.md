# Third Reich Documentary — Character Angle Pool

The 9 AM daily task picks **one random `pending` row** each morning (see the runbook for the exact
selection rule), scripts it, and marks it `scripted`. This replaced the old sequential
`episode-queue.md` on 2026-07-08 — that file is left on disk untouched but is no longer read by
the automation. Add/remove/edit rows anytime; the full menu of underlying material is in
`chapter-by-chapter-hidden-gems-report.md` and the raw text is `source-text-full-book.txt`.

**Status flow:** `pending` → `scripted` (rough script written) → `final` (you saved
script-final.md) → `produced` (mp4 rendered). A row can also bounce back to `pending` if the
scripter starts it and finds the material too thin (see runbook Step 3 fallback rule) — in that
case leave a one-line note in the Angle title cell noting it was attempted and skipped once.

**Types** — each drives a different research method and structure (full detail in the runbook):
- **FACET** — one life-dimension of one person
- **ARC** — one person's/pair's full multi-chapter throughline
- **RELATIONSHIP** — two people, alternating POV between them
- **INSTITUTION** — formation of a mechanism/body, story-first cold open, no single protagonist
- **EVENT** — a single dramatic incident, chapter-anchored (same shape as the old queue)
